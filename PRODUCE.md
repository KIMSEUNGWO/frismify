# Prismify - 프론트엔드 개발자를 위한 올인원 개발 도구

## 📋 프로젝트 개요

**Prismify**는 프론트엔드 개발자들을 위한 크롬 확장 프로그램입니다. "Prism"(프리즘)과 "-ify"의 조합으로, 웹페이지를 프리즘을 통해 들여다보듯 다양한 각도에서 분석하고 작업할 수 있다는 의미를 담고 있습니다.

### 🎯 핵심 목적

프론트엔드 개발자들이 자주 사용하는 도구들을 하나의 패키지로 통합하여, 개발 워크플로우를 최적화하고 생산성을 극대화합니다.

### 💡 개발 의도

- **개발자 도구 접근 최소화**: 브라우저 개발자 도구를 열지 않고도 대부분의 작업을 수행
- **단축키 중심 워크플로우**: 마우스 사용을 최소화하고 키보드 단축키로 빠른 작업 수행
- **모듈화된 플러그인 시스템**: 필요한 기능만 선택적으로 활성화하여 가벼운 사용 환경 제공
- **높은 확장성**: 플러그인 기반 아키텍처로 새로운 기능 추가 용이

## 🏗️ 아키텍처 설계

### 핵심 원칙: 추상화 (Abstraction)

Prismify의 가장 중요한 설계 원칙은 **추상화**입니다. 모든 플러그인은 표준화된 인터페이스를 따르며, 이를 통해:

- ✅ 플러그인 간 일관성 유지
- ✅ 신규 플러그인의 빠른 개발 및 통합
- ✅ 기존 플러그인의 손쉬운 수정 및 제거
- ✅ 확장 프로그램 전체의 안정성 향상

### 플러그인 개발 가이드라인

새로운 플러그인을 개발할 때는 다음의 추상화된 구조를 따릅니다. (현재 예시)

```tsx
interface Plugin {
  id: string;                    // 플러그인 고유 ID
  name: string;                  // 표시 이름
  description: string;           // 설명
  icon: string;                  // 아이콘
  shortcut: string;              // 기본 단축키
  enabled: boolean;              // 활성화 상태
  
  // 생명주기 메서드
  initialize(): void;            // 초기화
  activate(): void;              // 활성화
  deactivate(): void;            // 비활성화
  destroy(): void;               // 정리
  
  // 설정
  settings: PluginSettings;      // 플러그인별 설정
  updateSettings(settings: Partial<PluginSettings>): void;
}
```

## 🖥️ 사용자 인터페이스 구조

### 1. Popup (팝업 인터페이스)

**역할**: 빠른 접근 및 간단한 제어

**기능**:

- ✨ 플러그인 On/Off 토글
- 📊 현재 페이지 정보 표시 (해상도, 색상, 폰트)
- ⚡ 각 플러그인으로의 빠른 접근
- 🎨 시각적으로 구분되는 플러그인 카드

**특징**:

- 최소한의 UI로 빠른 조작 가능
- 단축키 표시로 학습 곡선 최소화
- 400px 고정 너비의 컴팩트한 디자인

### 2. Options (설정 페이지)

**역할**: 상세한 설정 및 커스터마이징

**기능**:

- 🎛️ 플러그인별 상세 설정
- ⌨️ 단축키 커스터마이징
- 🎨 테마 선택 (Prism, Purple Haze, Ocean Blue, Emerald)
- 📈 사용 통계 대시보드
- 💾 설정 내보내기/가져오기
- 🔄 초기화 및 백업 기능

**특징**:

- 사이드바 네비게이션으로 체계적 구성
- 섹션별 분리된 설정 (Dashboard, Tools, Shortcuts, Appearance 등)
- 실시간 미리보기 및 피드백

## 🔍 주요 경쟁 제품 분석

**현황**:

- 22,000명 이상의 개발자/디자이너가 사용

**주요 기능**:

- 요소 검사 (Inspector)
- 색상 추출
- 반응형 뷰어
- 에셋 추출 (이미지, SVG, 비디오)
- 기술 스택 분석
- 스크린샷

**단점**:

- 완전 유료 모델, 무료 기능 전혀 없음
- 활성화 제한 (3개 기기)
- "사기 같다"는 부정적 리뷰 존재

## ✨ Prismify의 핵심 차별점

### 1. **🏗️ 추상화 기반 아키텍처**

**경쟁사**:

- 기능이 하드코딩되어 있음
- 새 기능 추가 시 전체 코드 수정 필요

**Prismify**:

```tsx
// 플러그인 표준 인터페이스 - 예시
interface Plugin {
  id, name, shortcut
  initialize(), activate(), deactivate()
  settings, updateSettings()
}
```

- ✅ **커뮤니티**: 플러그인 마켓플레이스로 확장 가능
- ✅ **개발자 친화적**: 손쉽게 플러그인 개발 가능
- ✅ **확장성**: 손쉬운 기능 추가/제거
- ✅ **일관성**: 모든 플러그인이 동일한 방식으로 동작

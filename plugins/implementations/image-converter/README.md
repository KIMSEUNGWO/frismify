# Image Converter Plugin

이미지 파일 형식을 일괄 변환하는 플러그인입니다.

## 기능

- **다중 파일 지원**: 여러 이미지를 한 번에 선택하여 일괄 변환
- **드래그 앤 드롭**: 파일을 직접 드래그하거나 클릭하여 선택
- **지원 형식**:
  - PNG
  - JPEG/JPG
  - WebP
  - SVG (복사만 지원)
- **실시간 상태 표시**: 각 파일의 변환 상태를 실시간으로 확인
- **개별 다운로드**: 변환된 파일을 개별적으로 다운로드

## 사용 방법

1. 플러그인 활성화
2. 이미지 파일을 드래그하거나 클릭하여 선택
3. 변환할 형식 선택 (PNG, JPEG, WebP, SVG)
4. "일괄 변환 시작" 버튼 클릭
5. 변환 완료 후 개별 다운로드 버튼으로 파일 저장

## 기술 스택

- **Canvas API**: 이미지 형식 변환
- **FileReader API**: 파일 읽기
- **Blob URL**: 다운로드 링크 생성
- **Vue 3**: 반응형 UI

## 주의사항

- SVG → 다른 형식 변환은 Canvas API를 사용하여 래스터화됩니다
- 다른 형식 → SVG 변환은 지원하지 않습니다
- JPEG 변환 시 투명도는 흰색 배경으로 대체됩니다
- 변환 품질은 92%로 설정되어 있습니다
